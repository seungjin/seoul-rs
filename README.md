## How to write your content

1. Write your article in Markdown format with the following heading.  
```
$ cat john-smith-first-blog.md
+++
title = "John Smith's first blog"
date = "2026-03-02"
[taxonomies]
authors = ["John Smith", "John's Cat"]
tags = ["rust", "hello"]
+++

My first blog here.
```

2. Preview with zola
```
	$ zola serve
```
